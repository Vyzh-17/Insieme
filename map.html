Accessible Map for Disabled Persons

How to use this map

This interactive map shows locations with accessibility features for disabled persons. Filter by accessibility type or click on markers for more information.

Add a new accessible location

Or click on the map to add a marker at that location.

Accessibility Features

Wheelchair Accessible

Locations with ramps, wide doorways, and accessible restrooms.

Braille Accessible

Locations with braille signage, menus, or other tactile information.

Screen Reader Friendly

This map provides audio descriptions of locations when selected.
